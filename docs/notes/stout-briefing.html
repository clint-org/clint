02 / How it's built

The data model is pharma-shaped, not generic.

  • Structured records, not free text. Pipelines, assets, clinical trials, catalysts, and portfolios are stored as structured records. Not notes, not PDFs.
  • Events attach to assets. Every change — readouts, regulatory actions, approvals, label changes, deal activity — lands on the right asset as a typed, dated, source-cited record.
  • Visual hierarchy. Phase bars, color-coded markers, and a company-and-product hierarchy let a reader find the competitive read in seconds.
  • One data source, multiple consumers. The interface is one consumer of the data. Exports, briefing decks, client notices, and AI workflows are others. All read from the same source.
  • Citations on every event. Source URLs and dates are attached at ingest, so anything generated from the data can be traced back. This is what API access, MCP integration, and Clint-native AI all build on.

03 / What this changes for Stout

Five concrete shifts.

  • Live data. The competitive landscape on any asset or therapeutic area is current to the last event ingested. The strategic read sits on data that's current, not on a research sprint from two weeks ago.
  • AI workflows. Today, Stout analysts can export structured data from Clint into Claude, Copilot, or whatever AI tool the firm is using. On the roadmap: a tenant API and an MCP server so those tools can query Clint directly. After that: brief generation and scenario analysis built into Clint itself.
  • Whitelabel deliverable. Each client engagement can run as a Stout-branded Clint site at a custom domain. Agency, tenant, and per-engagement branding are all supported.
  • Engagement model. Today, recurring CI work is delivered as a quarterly deck. With Clint, the deliverable is a Stout-branded environment the client uses through the quarter, with Stout's analysis and recommendations layered into it.
  • Position. Evaluate, Citeline, and GlobalData are flat data sources. Other advisory firms still deliver CI as decks. No one in advisory is delivering CI as a whitelabel, AI-ready platform yet. Stout would be first.

07 / What buyers already complain about

The pain points buyers report — and where Clint sits against them.

Buyer complaints
  • Clunky interfaces. Said about all three big platforms.
  • Stale or inaccurate data. GlobalData by name. One commenter: "miss at least 20% of updated information if you trust just one platform."
  • Multi-platform redundancy. Buyers pay for two or more and manually cross-check.
  • Aggressive sales. GlobalData — "the most aggressive, obnoxious salesforce in our biz."
  • Opaque pricing. No published prices, every deal is bespoke.
  • Vendor instability. Cortellis has changed hands multiple times.
Where Clint sits against this
  • Coverage. Clint isn't pitched as a replacement for Evaluate or Citeline. It's where Stout's curated and annotated view sits on top of those sources, with the strategic read attached.
  • Interface. Buyers themselves describe the incumbents as clunky. That's a real opening, not marketing language.
  • Provenance. Source citations on every event. Required for output that gets attached to a fairness opinion or a board memo.
  • The combination. No incumbent offers whitelabel and AI-ready data together. That's the gap Clint fills.

08 / What the client deliverable looks like

Decks don't go away. They get shorter and more strategic.

The data review portion is already continuous in the platform, so the quarterly briefing becomes "here is what we recommend you do" instead of "here is what happened."

What's in the platform
  • Curation. Watchlist, competitor set, asset prioritization, therapeutic-area framing — Stout's domain expertise applied.
  • Annotation. Each meaningful event carries a Stout note: why it matters, what it shifts, what it implies.
  • Strategy. Scenario analysis, deal implications, recommendations — alongside the events that triggered them, all branded as Stout's work.
  • Switching cost. Watchlists, annotations, history, and scenario models live inside the Stout-branded environment. Leaving means losing institutional memory — which doesn't happen with static decks.
What changes for Stout
  • Cadence. Quarterly bursts become continuous presence in the client's workflow. Stout doesn't go dark for ten weeks at a time.
  • Briefings. Shorter and more strategic. The data review is already continuous in the platform, so the meeting is for recommendations and decisions.
  • Engagements. Recurring deck cadence is fragile — clients judge the relationship by one artifact per quarter. A live branded environment is harder to walk away from.
  • Margin mix. Junior assembly time compresses. Senior strategic time fills the engagement, and that's the higher-margin work.
