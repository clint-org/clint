Every marker falls under a category. Categories have specific markers. All markers in that category have the same or similar icons but maybe not filled, filled, or different pattern or colors. Like who we have it now. The rest of these are metadata that show on the tooltip when you hover over the marker. Categories include: 
    - Clinical Trial: Each product can be studied in multiple trials. Each trial is tied to one phase. Some trials can have combined phase 1 and 2, or combined phase 2 and 3. Each trial has a study start date and an study end date. We can show the NCT id next to the trial name on the grid column and tooltip when you hover over the marker for that trial on the timeline. This would provide more information about the trial and allow users to easily look up more details about it if they are interested. Every trial and phase will have a Primary Completion Date marker. This marker can change. 
    - Data: Data releases: topline, interim, full. Each data can be projected or actual. Each of these can be stout, company, primary, actual projection. These require a link to the source of the information, such as a news article, press release, or clinical trial registry entry and a description of the data being released.
    - Regulatory: Submission, FDA Acceptance. Each of these can be stout, company, primary, actual projection. For submission, can have three pathways: standard, priority, and cnpv; let's add this as a metadata tag. Sometimes regulatory markers can be tied to multiple trials/phases. So we should be able to assign that marker to multipel trials/phases. 
    - Approval: PDUFA date, launch date. Each of these can be stout, company, primary, actual projection. 
    - Loss of Exclusivity: Loe Date, generic entry date. Each of these can be stout, company, primary, actual projection. 


Each marker has the following metadata:
    - Title: a brief title for the event
    - projecttion: stout, company, primary, actual. Projected can be an open symbol, actual can be a filled symbol, and stout can be a different color or pattern. This would help to differentiate between projected and actual events, as well as the source of the information.
    - date: the date of the event or range of dates, or range into the future
    - description: a brief description of the event
    - link: a link to more information about the event (e.g. news article, press release, clinical trial registry, etc.)
