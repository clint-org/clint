Option C — marker-carried hollow cap (detail)

Larger view. The phase bar uses the real token (P3 teal #2dd4bf, fill 0.12, 1.2px stroke). An exact endpoint gets a plain dashed Trial Start/End marker and a hard bar edge. An approximate endpoint (month / year / quarter) additionally gets the hollow end-cap on the bar edge plus a ~caption.

How to read each row.
• SURMOUNT-3 — exact start, exact end: hard edges, no caps, no ~ (today's look, unchanged).
• REDEFINE-1 — ~month start: hollow cap + ~Nov '26 at the left edge; exact end stays a hard edge.
• ACACIA-HCM — ~year start and ~quarter end: hollow cap + ~2026 left, hollow cap + ~Q4 '27 right.
• TRIUMPH-1 — ongoing: right edge feathers (existing vocabulary) — visibly different from a hollow cap, so "ongoing" never gets confused with "approximate."
